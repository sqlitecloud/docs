## RENAME USER

### Syntax

RENAME USER **username** TO **new_username**

### Privileges

```
USERADMIN
```

### Description

The RENAME USER command is used to update an existing username to a new one.

### Return

OK

### Example

```bash
> RENAME USER user1 TO user2
OK
```
