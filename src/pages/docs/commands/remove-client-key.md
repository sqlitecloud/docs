## Syntax

REMOVE CLIENT KEY **keyname**

## Privileges

```
NONE
```

## Description

The REMOVE CLIENT KEY command is used to reset to a default value a <keyname>

## Return

OK

## Example

```bash
> REMOVE CLIENT KEY COMPRESSION
OK
```
