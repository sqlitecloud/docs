## Syntax

ENABLE USER **username**

## Privileges

```
USERADMIN
```

## Description

The ENABLE USER command re-enable a previously disabled user from the system.
Once re-enable, that username is able to login again.

## Return

OK

## Example

```bash
> ENABLE USER user1
OK
```
