## RENAME ROLE

### Syntax

RENAME ROLE **role_name** TO **new_role_name**

### Privileges

```
USERADMIN
```

### Description

The RENAME ROLE command is used to rename an existing role to a new name.

### Return

OK

### Example

```bash
> RENAME ROLE old_role TO new_role
OK
```
