## Syntax

SLEEP **ms**

## Privileges

```
NONE
```

## Description

The SLEEP command forces the current connection to sleep on the server-side for a specified amount of milliseconds.

## Return

OK

## Example

```bash
> SLEEP 100
OK (after 100ms)
```
