## SLEEP

### Syntax

SLEEP **ms**

### Privileges

```
NONE
```

### Description

The SLEEP command is used to force current connection to sleep on server-side for a specified amount of milliseconds.

### Return

OK

### Example

```bash
> SLEEP 100
OK (after 100ms)
```
